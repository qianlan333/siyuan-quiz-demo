# 小 IP 商业力测评 Demo

一个面向小 IP（想做引流 + 变现）的 5 维度商业力测评，25 题单选、5-7 分钟完成。

## 5 个维度

- 用户获取（流量引入能力）
- 用户维护（关系升温能力）
- 用户成交（临门一脚能力）
- 用户交付（预期管理能力）
- 用户裂变 / 升单（价值挖掘能力）

每维度 5 题，每个选项绑定一个"型"（如 用户成交 = 被动 / 压迫 / 推销 / 催眠）。

## 结果

- **总览**：综合分 / 等级 / 一句话画像 / 优势项 / 劣势项
- **5 维详细**：每维分数 + 主导型 + 特征 / 优势 / 风险 / 建议
- **推荐动作**：按劣势维度推服务 / 课程入口

## 平票规则

5 票分布只有一种平票场景（2-2-1-0），按每维固定优先级取（默认高分型优先），不出现"50% A + 50% B"的模糊结果。

## 本地运行

直接用浏览器打开 `index.html` 即可，纯静态、无需构建。
